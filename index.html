Uses quantum coherence to propose non-local MCMC moves. Provably mixes faster than classical MCMC for certain target distributions. Demonstrated on 27-qubit IBM processors.

Shows kernel values concentrate exponentially in the number of qubits for random circuits, undermining quantum advantage claims. Identifies conditions for avoiding concentration.

QCNNs mimic classical CNNs using alternating quantum convolution (local entangling) and pooling (partial measurement + feed-forward) layers. Efficient classically simulable special case.

2021 Algorithms
ADAPT-VQE: Adaptive Derivative-Assembled VQE
Grimsley, Economou, Barnes, Mayhall — 2021
Iteratively grows ansatz by selecting operators with largest gradient magnitude. Avoids barren plateaus via greedy circuit construction; achieves chemical accuracy with shallower circuits.

2009 Algorithms
HHL: Quantum Algorithm for Linear Systems
Harrow, Hassidim, Lloyd — PRL 2009
Solves $A|x\rangle=|b\rangle$ in $O(\log N \cdot \kappa^2/\epsilon)$ for sparse, well-conditioned $A$. Exponential speedup over classical; caveats on state preparation and readout.
Analysis arXiv
2019 Theory
Dequantization: Classical Algorithms for QML Tasks
Ewin Tang — 2019
Shows sampling-based classical algorithms achieve similar complexities to HHL for many ML tasks when given similar data access (quantum-inspired). Resets claims of quantum supremacy for ML.
arXiv
2010–20 NLP/LLM
DisCoCat: Compositional Distributional Quantum NLP
Coecke, Sadrzadeh, Clark — 2010; Meichanetzidis et al. — 2020
Categorical framework: grammar → string diagram → quantum circuit. Noun meanings = quantum states; verbs = quantum channels. Meaning composition = circuit composition.
Analysis
2019 Algorithms
QSVT: Quantum Singular Value Transformation
Gilyén, Su, Low, Wiebe — STOC 2019
Unifies quantum algorithms via block-encoding + quantum signal processing. Any polynomial $p(A/\alpha)$ can be applied to a block-encoded matrix. Subsumes HHL, phase estimation, amplitude amplification.
